# Claude Code 勉強会 — HTMLアプリ開発入門

---

## 概要

AIと会話しながらプログラムを作る体験をします。
今回は**サーバー不要・単一HTMLファイル**で動くアプリを題材にします。

---

## 必要なツール

### 1. Claude Code（必須）
AIにコードを書かせるメインツール。

**インストール**
```
npm install -g @anthropic-ai/claude-code
```

**起動方法**
```
claude
```

> Node.js (v18以上) が必要です。先にインストールしてください。
> https://nodejs.org/

### 2. Node.js（必須）
Claude Code の動作に必要。

- https://nodejs.org/ からLTS版をダウンロード
- インストール後、ターミナルで `node -v` と打って確認

### 3. VS Code（推奨）
コードを確認・編集するエディタ。

- https://code.visualstudio.com/
- Claude Code の VS Code 拡張機能もあります

### 4. ブラウザ（必須）
Chrome / Edge / Firefox など。HTMLファイルをダブルクリックして開くだけ。

## 当日の進め方

### ステップ1　環境確認（5分）

ターミナルを開いて確認：

```bash
node -v      # v18以上であること
claude       # Claude Code が起動すること
```

### ステップ2　作業フォルダを作る（2分）

```bash
mkdir my-app
cd my-app
claude
```

### ステップ3　AIに作ってもらう（メイン）

Claude Code に話しかけるだけ。例：

```
index.htmlを作って。ボタンを押すとサイコロを振って結果を表示するシンプルなアプリ
```

```
デザインをもっとかわいくして
```

```
1〜6の目が出た回数を記録するグラフも追加して
```

### ステップ4　ブラウザで確認

作られた `index.html` をダブルクリック → ブラウザで開いて動作確認。

### ステップ5　さらに改造

気に入らない点・追加したい機能をAIに伝えて改善する。

## Claude Code の基本的な使い方

| やること | 方法 |
|---------|------|
| ファイルを作る | 「〇〇.html を作って」と話しかける |
| 修正する | 「〇〇の部分を△△に変えて」と話しかける |
| コードを見る | VS Code でファイルを開く |
| 動作確認 | HTMLファイルをブラウザで開く |
| やり直す | 「さっきの変更を元に戻して」と話しかける |

## よくある質問

**Q. プログラムの知識がなくても大丈夫？**
大丈夫です。日本語で話しかけるだけでAIがコードを書きます。

**Q. 作ったものをネットに公開できる？**
できます。GitHub Pages や AWS S3 で無料〜低コストで公開可能（今回は扱いません）。

**Q. スマホでも使える？**
Claude Code はPC専用ですが、作ったHTMLはスマホのブラウザでも動きます。
